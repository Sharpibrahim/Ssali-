Beginner CSS Notes

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the color, layout, fonts, and positioning of elements on a webpage.

Note: CSS is essential for creating visually appealing websites.

Basic CSS Syntax

CSS follows the following structure:

selector {
    property: value;
}

Example:

h1 {
    color: blue;
}

Common CSS Properties

  • color: Sets the color of the text.
  • background-color: Sets the background color of an element.
  • font-size: Defines the font size of text.
  • padding: Adds space inside an element.
  • margin: Adds space outside an element.

Intermediate CSS Notes

Box Model

Every HTML element is a box. The Box Model consists of content, padding, border, and margin.

Note: Understanding the Box Model is crucial for managing layout and spacing.

Layout Properties

  • display: Controls how elements are displayed (e.g., block, inline, flex).
  • position: Controls the positioning of elements (e.g., static, relative, absolute).
  • float: Floats elements to the left or right, allowing text to wrap around them.

Flexbox

Flexbox is used for creating responsive layouts. Example:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
Note: Flexbox is particularly helpful for creating flexible and centered layouts.

Advanced CSS Notes

CSS Grid

CSS Grid is a powerful system for creating complex layouts. Example:

.grid-container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    gap: 10px;
}

Media Queries

Media queries allow you to apply different styles depending on screen size. Example:

@media screen and (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

CSS Variables

CSS variables allow for reusable values throughout your stylesheet. Example:

:root {
    --primary-color: #4CAF50;
}

.button {
    background-color: var(--primary-color);
}

CSS Animations

Animations make websites interactive and engaging. Example:

@keyframes example {
    0% { background-color: red; }
    100% { background-color: blue; }
}

.animate {
    animation: example 2s infinite;
}
